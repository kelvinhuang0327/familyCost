📚 如何獲取GitHub Personal Access Token

  1. 前往 GitHub Settings → Personal access tokens
  2. 點擊 "Generate new token (classic)"
  3. 設置權限：
    • ✅ repo (完整倉庫訪問)
    • ✅ workflow (更新GitHub Actions工作流程)
  4. 複製生成的token並貼到上方輸入框

🔒 安全說明

您的Token將使用AES-256-GCM加密儲存在本地，只有您能訪問。Token不會發送到任何第三方服務。
