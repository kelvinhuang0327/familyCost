🔧 Token儲存問題診斷工具

📋 診斷步驟

步驟 1: 檢查後端服務狀態
步驟 2: 測試Token API連接
步驟 3: 檢查瀏覽器網絡設置
步驟 4: 測試Token儲存

🛠️ 常見解決方案

方案 1: 重啟後端服務

在終端中執行以下命令：

# 停止現有服務 pkill -f "node server.js" # 重新啟動服務 node server.js

方案 2: 檢查端口佔用

檢查3001端口是否被其他程序佔用：

lsof -i :3001

方案 3: 清除瀏覽器緩存

清除瀏覽器緩存和Cookie，然後重新載入頁面

方案 4: 使用測試頁面

打開 test_token_save.html 進行詳細測試
