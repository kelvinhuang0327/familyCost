🔑 GitHub Token 格式檢查器

📋 如何獲取 GitHub Token

  1. 訪問 GitHub Token 設定頁面
  2. 點擊 "Generate new token" → "Generate new token (classic)"
  3. 填寫以下資訊：
    • Note: 家庭收支管理系統
    • Expiration: 90 days
    • Select scopes: 勾選 repo
  4. 點擊 "Generate token"
  5. 複製生成的 Token (只顯示一次！)

✅ 正確格式範例

ghp_1234567890abcdef1234567890abcdef12345678

❌ 常見錯誤

  • Token 太短 (少於 40 個字符)
  • 包含空格或特殊字符
  • 不是以 ghp_ 開頭
  • 權限不足 (需要 repo 權限)
